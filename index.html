eCALD Website Rebuild design system - WIP

Starts of a basic design system for the eCALD website rebuild. This is just a rough working document that will be further developed. The intention of this build is to use development principles that will allow us to build components with as little specificity as possible so that elements can be used as many contexts as possible.

Cards

Base cards

Each base card has:

  • Image or video
  • Heading
  • Text
  • Link locations (we can work out which elements are linked later)
  • Optional link or arrow button - positioned at the bottom of the card
  • Optional link with icon - positioned at the bottom of the card
  • Grid: 3, 2 or 1 column (1 column is full width)
